# 🛒 E-Commerce Store

A modern, responsive, and premium e-commerce website built with pure HTML, CSS, and JavaScript.

## ✨ Features

- 🏠 Modern Landing Page
- 📱 Fully Responsive Design
- 🔍 Product Search
- 🗂️ Category Filtering
- ⚡ Fast Performance
- 🛍️ Shopping Cart
- ❤️ Wishlist
- 🌙 Dark / Light Mode
- ⭐ Product Ratings
- 💸 Discount Badges
- 📦 Product Details
- 🔥 Best Sellers
- 🆕 New Arrivals
- 📞 Contact Section
- 📍 Store Location
- 💳 Modern Checkout UI
- 🎨 Smooth Animations
- ♿ Accessible Design

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage
- Font Awesome
- Google Fonts
